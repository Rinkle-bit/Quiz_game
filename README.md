🌐 WebCraft Quiz Arena
An interactive and visually appealing Web Development Quiz Game built using **HTML**, **CSS**, and **JavaScript**.
This project helps developers improve their frontend knowledge through multiple-choice quizzes with gamification features like XP, leaderboard, streak tracking, timer mode, and multiple difficulty levels.

---
🎮 How to Play
1. Select a Quiz Category.
2. Choose Difficulty Level.
3. Select Number of Questions.
4. Enable/Disable Timer Mode.
5. Click **Start Round**.
6. Answer each question.
7. Build streaks to increase score.
8. Earn XP and improve your rank.
